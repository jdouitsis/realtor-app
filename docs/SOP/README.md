# Standard Operating Procedures

This directory contains SOPs for common repository-level tasks.

## Index

|-----|-------------|

- **Setting up a new tool** (linter, formatter, test runner) → `sop-turbo-pack.md`
- **Modifying git hooks** → `sop-turbo-pack.md`
